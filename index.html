📦 Variáveis e Tipos de Dados

Explicação

Variáveis armazenam dados em memória e permitem nomear valores para uso posterior. Em JavaScript usamos let (variável mutável), const (valor fixo/imutável) e var (escopo de função — pouco usado atualmente). Tipos comuns: string (texto), number (números), boolean (verdadeiro/falso), null, undefined e objetos/arrays para estruturas mais complexas.

Exemplo de Código

let nome = "João";
let idade = 25;
const ativo = true;
console.log(nome, idade, ativo);

O que esse código faz: declara duas variáveis (nome e idade) e uma constante (ativo). Em seguida imprime esses valores no console. Serve para demonstrar declaração, atribuição e exibição de valores.

📋 Arrays

Explicação

Arrays são listas ordenadas que armazenam múltiplos valores acessíveis por índice (começando em 0). Permitem operações como adicionar, remover, iterar e transformar elementos. Úteis para coleções de dados, filas, pilhas e tabelas simples.

Exemplo de Código

let frutas = ["maçã","banana","laranja"];
frutas.push("uva");
console.log(frutas);

O que esse código faz: cria um array com três strings, usa push para adicionar "uva" ao final e imprime o array atualizado. Demonstra criação, modificação e exibição de arrays.

➕ Operadores Matemáticos

          

            
Explicação

            

              Operadores matemáticos realizam cálculos básicos e avançados: + (adição), - (subtração), * (multiplicação), / (divisão), % (resto da divisão) e ** (exponenciação). Combine-os com variáveis para expressar algoritmos e fórmulas.
            

          

          

            
Exemplo de Código

            
let a = 10, b = 3;
console.log(a + b, a - b, a * b, a / b, a % b, a ** b);

            
O que esse código faz: define dois números e imprime os resultados de soma, subtração, multiplicação, divisão, resto e potência. Útil para ver o efeito de cada operador.

⚖️ Operadores Relacionais

          

            
Explicação

            

              Operadores relacionais comparam valores e retornam booleanos. Use ==/!= para comparação flexível, ===/!== para comparação estrita (tipo + valor), e >, <, >=, <= para comparações numéricas ou alfabéticas. Essenciais para decisões e filtros.
            

          

          

            
Exemplo de Código

            
let x = 10, y = 5;
console.log(x > y);  // true
console.log(x === y); // false
console.log(x != y);  // true

            
O que esse código faz: compara dois números e imprime resultados booleanos para maior que, igualdade estrita e desigualdade. Ajuda a entender como cada operador avalia os valores.

          

        

      


      
      

        

          
🔗 Operadores Lógicos

          

            
Explicação

            

              Operadores lógicos combinam expressões booleanas: && (E) retorna true se ambos forem true; || (OU) retorna true se ao menos um for true; ! (NÃO) inverte o valor booleano. Usados para compor condições mais complexas.
            

          

          

            
Exemplo de Código

            
let idade = 20;
let possuiCarteira = true;
if (idade >= 18 && possuiCarteira) { console.log("Pode dirigir"); } 
else { console.log("Não pode dirigir"); }
console.log(!possuiCarteira); // false

            
O que esse código faz: verifica se a pessoa tem idade suficiente e carteira (operador &&) para decidir se pode dirigir, e usa ! para inverter o valor de possuiCarteira. Demonstra combinação e negação de condições.

          

        

      


      
      

        

          
🔀 Estruturas Condicionais (If/Else)

          

            
Explicação

            

              If/else permite executar blocos diferentes dependendo do resultado de uma condição booleana. Combine if, else if e else para tratar múltiplos caminhos. São fundamentais para controlar fluxo e reagir a dados.
            

          

          

            
Exemplo de Código

            
let idade = 18;
if(idade >= 18){ console.log("Maior de idade"); }
else { console.log("Menor de idade"); }

            
O que esse código faz: testa se a variável idade é maior ou igual a 18. Se for, imprime "Maior de idade", caso contrário imprime "Menor de idade". Demonstra decisão binária.

🎯 Estrutura Switch/Case

          

            
Explicação

            

              Switch/case é útil para testar uma mesma expressão contra múltiplos valores (casos) de forma legível. Cada case é testado sequencialmente até encontrar uma correspondência; use break para evitar execução contínua.
            

          

          

            
Exemplo de Código

            
let dia = 3;
switch(dia){
  case 1: console.log("Domingo"); break;
  case 2: console.log("Segunda"); break;
  case 3: console.log("Terça"); break;
  default: console.log("Outro dia"); }

            
O que esse código faz: avalia a variável dia e executa o bloco correspondente ao case igual a 3, imprimindo "Terça". Se nenhum case corresponder, executa default. Mostra como mapear valores para ações.

          

        

      


      
      

        

          
🔄 Estrutura de Repetição While

          

            
Explicação

            

              while repete um bloco enquanto uma condição for verdadeira. Use quando o número de iterações não é conhecido antecipadamente. Evite loops infinitos garantindo que a condição possa se tornar falsa.
            

          

          

            
Exemplo de Código

            
let i = 0;
while(i < 5){
  console.log(i);
  i++;
}

            
O que esse código faz: inicia i em 0 e imprime valores de 0 a 4, incrementando i a cada iteração. Demonstra controle de repetição com condição de parada.

          

        

      


      
      

        

          
🔁 Estrutura de Repetição For

          

            
Explicação

            

              for é ideal quando se conhece o número de repetições. Possui inicialização, condição e incremento, tudo na mesma linha, tornando-o compacto para percorrer índices ou executar tarefas repetidas.
            

          

          

            
Exemplo de Código

            
for(let i=0;i<5;i++){
  console.log(i);
}

            
O que esse código faz: repete um bloco 5 vezes, imprimindo os valores de 0 a 4. Mostra a estrutura típica de inicialização, condição e incremento do for.

          

        

      


      
      

        

          
🔃 Estrutura de Repetição Do While

          

            
Explicação

            

              do/while executa o bloco pelo menos uma vez e depois verifica a condição para repetir. Útil quando é necessário que o código execute antes de avaliar a condição (por exemplo, ler entrada do usuário).
            

          

          

            
Exemplo de Código

            
let i=0;
do{
  console.log(i);
  i++;
}while(i<5);

            
O que esse código faz: imprime os números de 0 a 4 garantindo que o bloco rode pelo menos uma vez antes de verificar a condição.

          

        

      


      
      

        

          
🔂 Estrutura de Repetição ForEach

          

            
Explicação

            

              forEach é um método de array que executa uma função para cada elemento. É ideal para iterações simples quando não se precisa de um índice explícito ou de interromper o loop (não suporta break).
            

          

          

            
Exemplo de Código

            
let numeros=[1,2,3];
numeros.forEach(n=>console.log(n));

            
O que esse código faz: percorre o array numeros e imprime cada elemento usando uma arrow function. Mostra iteração simples sem manipular índices.

          

        

      


      
      

        

          
⚙️ Funções em JavaScript

          

            
Explicação

            

              Funções encapsulam lógica reutilizável; podem receber parâmetros e retornar valores. Existem declarações tradicionais, expressões de função e arrow functions. Boas funções têm responsabilidade única e nomes descritivos.
            

          

          

            
Exemplo de Código

            
function saudacao(nome){
  return "Olá, "+nome+"!";
}
console.log(saudacao("João"));

            
O que esse código faz: define uma função que recebe um nome e retorna uma saudação. Em seguida chama a função com "João" e imprime a mensagem. Demonstra entrada, processamento e retorno.
