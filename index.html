⚖️ Operadores Relacionais

          

            
Explicação

            

              Comparam valores: ==, ===, !=, !==, >, <, >=, <=.
            

          

          

            
Exemplo de Código

            

              
let x = 10, y = 5;
console.log(x > y);  // true
console.log(x === y); // false
console.log(x != y);  // true
            

          

        

      




      
      

        

          
🔗 Operadores Lógicos

          

            
Explicação

            

              Combinam condições: && (E), || (OU), ! (NÃO).
            

          

          

            
Exemplo de Código

            

              

let idade = 20;
let possuiCarteira = true;
if (idade >= 18 && possuiCarteira) {
  console.log("Pode dirigir"); 
} else { console.log("Não pode dirigir");
}
console.log(!possuiCarteira); // false

🎯 Estrutura Switch/Case

          

            
Explicação

            

              Permite testar múltiplos casos de uma variável.
            

          

          

            
Exemplo de Código

            

              

let dia = 3;
switch(dia){
  case 1: console.log("Domingo"); break;
  case 2: console.log("Segunda"); break;
  case 3: console.log("Terça"); break;
  default: console.log("Outro dia"); }
              

          

        

      




      
      

        

          
🔄 Estrutura de Repetição While

          

            
Explicação

            

              Executa um bloco enquanto a condição for verdadeira.
            

          

          

            
Exemplo de Código

            

              

let i = 0;
while(i < 5){
  console.log(i);
  i++;
}
              

          

        

      




      
      

        

          
🔁 Estrutura de Repetição For

          

            
Explicação

            

              Executa um bloco um número definido de vezes.
            

          

          

            
Exemplo de Código

            

              

for(let i=0;i<5;i++){
  console.log(i);
}
              

          

        

      




      
      

        

          
🔃 Estrutura de Repetição Do While

          

            
Explicação

            

              Executa o bloco pelo menos uma vez e repete enquanto a condição for verdadeira.
            

          

          

            
Exemplo de Código

            

              

let i=0;
do{
  console.log(i);
  i++;
} while(i<5);
              

          

        

      




      
      

        

          
🔂 Estrutura de Repetição ForEach

          

            
Explicação

            

              Itera sobre cada elemento de um array.
            

          

          

            
Exemplo de Código

            

              

let numeros=[1,2,3];
numeros.forEach(n=>console.log(n));
